Potato-Assistant

Rebalancing tool for couch-potato investors

View project onGitHub

About.

Potato-Assistant is a simple rebalancing tool for passive investing, also known as the "couch-potato strategy". It has been inspired by the canadian couch potato blog.

Potato-Assistant is free software, which means you can use it, modify it and distribute it with no restrictions. You can download the source code from GitHub. It is distributed under the GPL license.

Features.

    Using Potato-Assistant you can:
  • Define your own portfolio model or use a predefined template.
  • Visualize the current state of your portfolio and how much it deviates from your model.
  • See where to invest to keep your funds balanced.
  • See how to rebalance your funds.
